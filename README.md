# Sistem Pemilihan dan Pemantauan Ngantuk untuk Optimalisasi Industri pada Jam Kerja
<div align="justify">

Program ini bertujuan untuk memantau aktivitas kerja pengguna menggunakan kamera (deteksi wajah) sambil menyediakan aplikasi menggambar (Paint) sebagai antarmuka interaktif.

Program akan:
Mendeteksi wajah menggunakan kamera.
Mengatur siklus Kerja → Istirahat → Persiapan Kembali secara otomatis.
Menampilkan status, waktu kerja, peringatan, dan tampilan kamera pada layar Paint.
Menyimpan hasil gambar pengguna bila diinginkan.
